## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-init`
- `allow-speak`
- `allow-stop`
- `allow-pause`
- `allow-resume`
- `allow-set-rate`
- `allow-set-pitch`
- `allow-set-voice`
- `allow-set-primary-lang`
- `allow-get-voices`
- `allow-get-voice-id`
- `allow-get-all-voices`
- `allow-get-granularities`
- `allow-get-speaking-lang`
